Title Belt: Grid Module Demo

This is a simple demo to showcase the capabilities of what I consider a really simple modern grid syntax. I built this framework to allow me to think more logically about how I lay things out on my pages while enforcing a mobile-first approach to my content.

My approach doesn't use floats. I don't believe floats are necessary for basic layout concerns, so instead I use inline-block elements where the .container specifies a font-size: 0 and then the .col class uses font-size: initial to reset things as needed.

Dynamic Column Overview

You can supply custom breakpoint layout reflows by applying an attribute to your column of mobile="...", tablet="...", desktop="...". Since you're supplying a specific property for the given column at a given context, I believe it makes more sense as an attribute as opposed to a class as other frameworks do. 


<div class="container">
  <div class="col vbaseline blue" mobile="whole" tablet="half" desktop="third">Bacon ipsum dolor amet...</div>
  <div class="col vbaseline magenta" mobile="whole" tablet="half" desktop="fourth">Bacon ipsum dolor amet...</div>
  <div class="col vbaseline seafoam" mobile="whole" tablet="half" desktop="third">Bacon ipsum dolor amet...</div>
  <div class="col vtop violet" mobile="whole" tablet="half" desktop="fourth">Bacon ipsum dolor amet...</div>
  <div class="col vtop yellow" mobile="whole" tablet="half" desktop="fifth">Bacon ipsum dolor amet...</div>
  <div class="col vtop green" mobile="whole" tablet="half" desktop="fourth">Bacon ipsum dolor amet...</div>
</div>
